## 🎯 О проекте и цель разработки
Этот проект — мой персональный **Pet-проект**, созданный с целью глубокой отработки навыков семантической верстки, работы с комплексными CSS-сетками и адаптивности. 

За основу был взят оригинальный татарстанский веб-ресурс. Задача заключалась в том, чтобы **полностью с нуля, своими руками**, воссоздать визуальную копию (реплику) интерфейса, используя современные стандарты веб-разработки, без использования готовых UI-фреймворков (Bootstrap, Tailwind и т.д.).

## 🛠️ Какие навыки я прокачал на этом проекте
*   **Архитектура и анализ UI:** Провел декомпозицию существующего сайта на независимые блоки, выстроил логичную HTML-структуру.
*   **Pixel-Perfect верстка:** Добился точного визуального соответствия оригиналу (шрифты, отступы, сетка элементов).
*   **Адаптивность (Responsive Design):** Реализовал гибкую адаптивную сетку, чтобы сайт корректно и красиво отображался как на FullHD мониторах, так и на экранах мобильных телефонов.
*   **Оптимизация ресурсов:** Самостоятельно оптимизировал и подключил графику, иконки и кастомные шрифты для ускорения загрузки страниц.

## 💻 Технологический стек
*   **HTML5** (Строгая семантическая верстка для лучшего SEO и доступности)
*   **CSS3** (Использование Flexbox и Grid Layout для построения сложных сеток, CSS-переменные для удобного управления стилями)
*   **Оптимизация:** SVG-графика, Google Fonts.

## 🚀 Инструкция по локальному запуску

Чтобы сайт открывался корректно (без проблем с путями к картинкам и шрифтам), его лучше запускать через локальный веб-сервер, а не просто двойным кликом по файлу. Выберите любой удобный способ:

### 1. Подготовка (Клонирование репозитория)
Откройте терминал и выполните команды:
```bash
# Клонируем проект
git clone [https://github.com/BibaPirat6/espanos-site.git](https://github.com/BibaPirat6/espanos-site.git)

# Переходим в папку проекта
cd espanos-site

Если ты уже работаешь в Visual Studio Code:

1. Установи расширение **Live Server** (автор: Ritwick Dey).
   - Открой VS Code → вкладка Extensions (`Ctrl+Shift+X`)
   - В поиске введи `Live Server` → нажми **Install**
2. Кликни правой кнопкой мыши по файлу `index.html`
3. Выбери **"Open with Live Server"**
4. Сайт автоматически откроется в браузере по адресу `http://127.0.0.1:5500`
